02b — Color Role Rules — Elevated Southwest
Turquoise — when to use
  • ✓  Primary CTA buttons (.btn-primary)
  • ✓  Text links and hover states
  • ✓  Quiz option selected state
  • ✓  Progress bar fill
  • ✓  Interactive chevrons / arrows
  • ✗  Eyebrows (use adobe)
  • ✗  Card border accents (use adobe)
  • ✗  Decorative icons (use adobe)
Adobe — when to use
  • ✓  All section eyebrows
  • ✓  Card top border accents (3px)
  • ✓  Trust/guarantee icons & badges
  • ✓  Savings pills (testimonials)
  • ✓  Section labels (WHAT YOUR CPA WILL ASK)
  • ✓  Decorative quote marks
  • ✗  CTA buttons (use turquoise)
  • ✗  Interactive selections (use turquoise)
Design intent: The ivory/sand background does the heavy lifting for the southwest warmth. Adobe accents give it character and humanity. Turquoise is reserved as the single action signal — making CTAs pop because they're rare.
